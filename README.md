# Tutorial Básico de Streamlit

Este es un tutorial interactivo multipágina diseñado para aprender los fundamentos de Streamlit.

## Requisitos

Asegúrate de tener instalado Streamlit:

```bash
pip install -r requirements.txt
```

## Cómo ejecutar

Para iniciar el tutorial, ejecuta el siguiente comando en tu terminal desde la carpeta `actividad2_ciencia_datos`:

```bash
streamlit run Inicio.py
```

## Contenido

1.  **Inicio**: Introducción y bienvenida.
2.  **Texto y Visualización**: Aprende a mostrar títulos, texto formateado, código, fórmulas y métricas.
3.  **Entradas y Botones**: Interactúa con botones, sliders, cajas de texto y selectores.
4.  **Diseño y Columnas**: Organiza tu aplicación con columnas, pestañas y expansores.
5.  **Estado de Sesión**: Entiende cómo persistir datos entre interacciones usando `session_state`.
6.  **Formularios**: Aprende a agrupar inputs y procesarlos en lote.
7.  **Multimedia**: Sube archivos y muestra imágenes, audio y video.
8.  **Status y Feedback**: Usa barras de progreso, spinners y notificaciones.
9.  **Gráficos Simples**: Visualiza datos básicos sin necesidad de librerías externas.

¡Disfruta aprendiendo!
